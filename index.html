🚀 Git Form Builder

Erstellen Sie professionelle Formulare direkt in Ihrem Git Repository

Hinweis: Nach dem Erstellen Ihres Formulars können Sie es mit dem "Formular speichern" Button speichern. Die Formulardefinition wird direkt in diese HTML-Datei eingebettet. Committen Sie die Datei anschließend in Ihr Git Repository.

🔽 Dropdown-Menüs: Bei Dropdown-Fragen können Sie Optionen aus Textdateien (.txt oder .csv) laden. Erstellen Sie eine Datei mit einer Option pro Zeile. Kommentare mit # am Zeilenanfang werden ignoriert.
